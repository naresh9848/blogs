## What is Markdown?

Markdown is a lightweight markup language created by John Gruber in 2004. The goal was simple: write plain text that reads naturally and converts cleanly to HTML.

## Basic Syntax Cheat Sheet

### Text Formatting

```
**bold text**
*italic text*
~~strikethrough~~
`inline code`
```

### Headings

Use `#` for headings. One `#` is `<h1>`, two `##` is `<h2>`, and so on.

### Links and Images

```markdown
[Link text](https://example.com)
![Alt text](image.png)
```

### Lists

Unordered:
- Item one
- Item two
- Item three

Ordered:
1. First step
2. Second step
3. Third step

### Blockquotes

> Great writing is clear thinking made visible.

### Code Blocks

Use triple backticks with an optional language name:

```javascript
const greet = (name) => `Hello, ${name}!`;
console.log(greet("world"));
```

## Tips for Blog Writing

**Keep sentences short.** Readers skim. Short sentences scan better.

**Use headings generously.** They help people navigate long posts.

**One idea per paragraph.** Give each thought its own space to breathe.
